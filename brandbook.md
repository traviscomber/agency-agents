# AgencyOS Brandbook

## Brand position
- AgencyOS is a specialist workspace for structured AI work.
- The interface should feel calm, precise, and operational.
- Avoid generic chatbot vibes, overly playful color, and ornamental noise.

## Visual direction
- Use a neutral base with a restrained slate-indigo accent.
- Prefer white surfaces, soft borders, and strong typographic hierarchy.
- Use dark panels only for emphasis, not as a default theme.
- Keep spacing generous and section rhythm clear.

## Typography
- Headlines should be confident and compact.
- Body copy should be readable, direct, and slightly editorial.
- Labels and meta text should be smaller, lighter, and consistent.

## Component rules
- Cards should have clear purpose and one obvious action.
- Badges should communicate status, plan, or division.
- Primary CTAs should be visually distinct without feeling loud.
- Use rounded pills for filters and tags, not heavy rectangles.

## Interaction rules
- Keep hover states subtle and meaningful.
- Use motion sparingly for page transitions and hierarchy changes.
- Every screen should answer: what is the next action?

## Copy rules
- Use direct, task-oriented language.
- Prefer verbs: run, save, compare, review, upgrade.
- Avoid hype, filler, and abstract marketing language.

## Application notes
- Landing pages should create context and trust fast.
- Library pages should be easy to scan and compare.
- Detail pages should present structure, inputs, and outputs clearly.
